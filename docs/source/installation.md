# Installation

**How to install SPDB in your Python project.**

## Requirements

- Python 3.10 or newer
- SharePoint access (username and password)
- Poetry (recommended) or pip

## Install with Poetry

```sh
poetry add spdb
```

## Install with pip

```sh
pip install spdb
```

## Dependencies

SPDB uses:

- Pydantic v2
- Office365-REST-Python-Client
- Ruff (for linting)
- Pytest (for testing)
- Pre-commit (for code quality)
